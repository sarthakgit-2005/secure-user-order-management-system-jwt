# Secure User Order Management System

A secure backend REST API built using Spring Boot, Spring Security, and JWT Authentication to manage users and orders.

This project demonstrates secure API development, layered architecture, authentication, and exception handling.

---

## Features

- User Registration
- Login using JWT Authentication
- Secure REST APIs
- Order Management
- Admin Management APIs
- Global Exception Handling
- DTO based Request & Response
- Layered Architecture

---

## Technologies Used

- Java
- Spring Boot
- Spring Security
- JWT (JSON Web Token)
- Spring Data JPA
- Hibernate
- Maven
